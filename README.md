# BoredBot(borredbot.netlify.app)

BoredBot is a fun and interactive mini web app that helps you find random activities whenever you're bored. It uses the Bored API to fetch activity ideas and updates the page with animations, gradients, and a playful bouncing effect.

## Features
- Fetches a random activity from the Bored API
- Changes background to a random gradient on each fetch
- Updates the title dynamically
- Displays the activity in a bouncing, moving animation
- Clean and simple UI with Google Fonts

## Technologies Used
- HTML
- CSS (Flexbox, gradients, animations)
- JavaScript (DOM manipulation, Fetch API)
